# AgriSphere 360

AgriSphere 360 is an enterprise-level precision farming web platform designed to provide farmers, agronomists, and administrators with a comprehensive suite of tools for farm management, AI-driven advisory, and market intelligence.

## 🚀 Features

- **Precision Farm Management**: Track and manage crop details, active lands, and historical farm data.
- **AI-Driven Advisory**: Ask Google Gemini natural language questions regarding your soil condition, crop health, or pest treatments and get real-time recommendations.
- **Interactive Dashboard**: Get a comprehensive overview of your net profits, total farms, system alerts, and quick actions tailored explicitly for agriculture workflows.
- **Financial Analytics**: Track and visualize income and expenses efficiently over time.
- **Live Market Prices**: Track "mandi" prices to compare active trends for your primary crops like Wheat, Rice, and Cotton.
- **Global State Profile Settings**: Customize user preferences seamlessly across the entire application using Context API and localStorage persistence.

## 🏗 Tech Stack

- **Frontend Framework**: React
- **Build Tool**: Vite
- **Routing**: React Router DOM
- **UI Icons**: Lucide React
- **Charting**: Recharts
- **Styling**: Vanilla CSS (Fluid Glassmorphism & Custom Global Themes)
- **Local Database**: Native Browser localStorage
- **AI Integration**: Prepared for Google Gemini / Vertex AI

## ⚙️ How to Run Locally

1. Clone or download this repository.
2. Navigate to your project folder using your terminal:
   ```bash
   cd agrisphere_360_web
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your web browser and navigate to the provided link (usually `http://localhost:5173`).
